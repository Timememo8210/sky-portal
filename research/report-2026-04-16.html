🔄 M365 Copilot与Copilot Chat重大改名，4月15日生效

Microsoft于4月15日正式更新M365 Copilot和Copilot Chat的命名体系，将产品名称与性能层级绑定。Licensed用户将看到完整版Copilot体验，而未授权用户将看到"M365 Copilot (Basic)"版本及升级提示。Word、Excel、PowerPoint中的Copilot界面也同步更新，免费用户首次可以直接在应用内体验基础AI功能。

Microsoft officially updated the naming of M365 Copilot and Copilot Chat on April 15, tying product names to performance tiers. Licensed users see the full Copilot experience while unlicensed users see "M365 Copilot (Basic)" with upgrade prompts. Word, Excel, and PowerPoint Copilot interfaces were also updated, allowing free users to experience basic AI features in-app for the first time.

来源：Microsoft Directions on Microsoft / Reddit r/msp

🛡️ Security Copilot将纳入M365 E5，4月20日起生效

从4月20日起，Microsoft将把Security Copilot加入Microsoft 365 E5许可证。这意味着E5用户无需额外付费即可使用AI驱动的安全分析和威胁检测功能。此举大幅提升了E5许可证的价值，也表明Microsoft正将AI安全能力从附加产品转向核心组件。对IT管理员来说，这将简化部署流程并降低总体安全成本。

Starting April 20, Microsoft is adding Security Copilot to the M365 E5 license at no extra cost. E5 users gain AI-driven security analytics and threat detection. This significantly boosts E5 value and signals Microsoft's shift of AI security from add-on to core component. For IT admins, this simplifies deployment and reduces overall security costs.

来源：Reddit r/msp / Microsoft Partner Center

💡 实用技巧：Copilot多模型切换功能

Microsoft Copilot平台最新升级支持在多个AI模型间切换，包括OpenAI GPT系列和Anthropic Claude。在Copilot Chat中点击模型选择器即可切换，不同模型在不同任务上表现各异——Claude擅长长文档分析，GPT则在创意写作和数据处理上更优。建议根据具体工作场景选择最适合的模型。

The latest Copilot upgrade supports switching between multiple AI models including OpenAI GPT and Anthropic Claude. Click the model selector in Copilot Chat to switch. Different models excel at different tasks—Claude for long document analysis, GPT for creative writing and data processing. Choose the best model for your specific workflow.

来源：Microsoft Release Notes
